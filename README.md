# OrchestrateOS Marketplace

**76 production-ready Claude Code plugins.** Skills that execute, not just respond.

## Installation

```bash
/plugin marketplace add unmistakablecreative/orchestrateos-marketplace
```

Then install any plugin:

```bash
/plugin install blog-outline@orchestrateos
/plugin install deep-research@orchestrateos
/plugin install slide-designer-premium@orchestrateos
```
